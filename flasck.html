Sobre o flasck

Sendo ele escrito em Python e disponível sobre a licença de código aberto, o Flask é um micro-framework com uma multiplataforma que provê um modelo básico para o desenvolvimento web. Ele é destinado principalmente a pequenas aplicações com requisitos mais simples, como por exemplo, a criação de um site básico. Possui um núcleo simples e expansível que proporciona que um projeto possua apenas os recursos necessários para sua execução.

Algumas características sobre o Flasck

✔ Como ponto principal, o flasck é simples, pois possui apenas o necessário para o desenvolvimento de uma aplicação, um projeto escrito com Flask é mais básico se comparado aos frameworks maiores, já que a quantidade de arquivos é muito menor e sua arquitetura é muito mais simples.

✔ É rápido no desenvolvimento, no Flask, o desenvolvedor se ocupa apenas em desenvolver o que precisa para um projeto, sem a necessidade de realizar configurações que muitas vezes não são utilizadas.

✔ Seus projetos são menores, por possuir uma arquitetura muito simples os projetos escritos em Flask tendem a ser menores e mais leves se comparados a frameworks maiores.
